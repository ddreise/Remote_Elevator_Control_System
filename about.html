Project VI - About

Welcome! This is the about page for our team's Project VI!
This is where you will find an overview of our project. The goal of this project is to design an elevator using 3 STM32 Nucleo devices connected to a Raspberry Pi in a CAN network. We will use a defined protocol for managing the CAN network as well as a custom gui interface to the Raspberry Pi.
We are very excited to show you our work!
